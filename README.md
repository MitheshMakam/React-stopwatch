# ⏱ React Stopwatch

A responsive and interactive stopwatch built using React.

## 🚀 Features
- Start / Stop timer
- Reset functionality
- Accurate time tracking
- Clean and minimal UI

## 🛠 Tech Stack
- React (useState, useEffect)
- JavaScript (ES6+)
- CSS
